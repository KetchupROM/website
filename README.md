# KetchupROM

KetchupROM is a custom Android ROM project inspired mainly by Cyanogenmod and LineageOS.
This repository hosts the **KetchupROM GitHub Pages site**, including the homepage and downloads page.

## Features

- Lightweight and optimized for legacy and generic devices
- Includes only essential apps ( Phone, Messages, Contacts, Terminal and File Manager. Yes, thats all of the total apps. )
- Desktop-first web interface for downloads and information
- Open source and community-driven
